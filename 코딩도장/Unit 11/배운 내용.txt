1. 시퀀스 자료형: 값이 연속적으로 이어진 자료형
  - 리스트, 튜플, range, 문자열, bytes, bytearray
  - 공통 기능
    - in: 시퀀스 객체에 특정 값이 있는가 (<-> not in)
    - +: 객체 연결 -> 새 시퀀스 객체 생성
      - range는 불가
    - *: 시퀀스 객체 반복
      - range는 불가
  
2. len(): 요소의 개수
  - 문자열은 공백까지 포함, 따옴표는 제외
  - UTF-8 문자열이 차지하는 실제 바이트 수: len(문자열.encode('utf-8'))
    
3. sequence[i]: 시퀀스 객체 sequence의 인덱스가 i인 요소에 접근
  - 인덱스는 0부터 시
  == sequence.__getitem__(i)
  - 음수 인덱스: 뒤에서부터 접근
  - 인덱스 범위에서 벗어나면 IndexError 발생
4. 요소에 값 할당: sequence[i] = a
  - 튜플, range, 문자열은 불가
5. del sequence[i]: 요소 삭제
  - 튜플, range, 문자열은 불가

6. 슬라이스: 시퀀스 객체의 일부를 잘라냄
  : sequence[i:j]
  - 끝 인덱스는 가져오려는 범위에 포함되지 않음 -> j-1 인덱스까지
  - 음수로도 지정 가능
  - 인덱스 증가폭 지정 가능: sequence[i:j:k] -> 인덱스 i부터 k씩 증가시키면서 인덱스 j까지 가져옴
    - k가 음수이면 인덱스를 감소시키면서 요소를 가져옴
  - 인덱스 생략 가능
    - 처음부터 j까지: [:j]
    - i부터 끝까지: [i:]
    - 전체: [:], [::]
  - 슬라이스 객체: slice(시작 인덱스 i, 끝 인덱스 j, 증가폭 k)
  - 슬라이스에 요소 할당, 삭제 가능
    - 튜플, range, 문자열은 불가
