1. if 조건식:
        코드1
   else:
        코드2
2. 조건부 표현식: x = a1 if condition else a2
  - condition이 True면 x에 a1 할당, 아니면 a2 할당

3. 조건문
  - None을 넣으면 False로 취급
  - 0 이외에는 참
  - 빈 문자열은 거짓, 아니면 참
  -> not None / not 0 / not '' 는 참
  - 빈 리스트, 튜플, 딕셔너리, 세트는 False
  
4. 조건식 여러 개 지정하기
  - 논리 연산자 이용
  - 중첩 조건문
  - 부등호 연달아 사용 가능
