1. 변수의 이름
  - 영문문자 & 숫자
  - 대소문자 구분
  - 숫자로 시작 불가, _로 시작 가능
  - 특수문자 사용 불가
  - 키워드(if, for, while, and, or 등) 사용 불가
2. 변수의 자료형 알아내기: type()
3. 변수에 값 할당하기
  - 변수 여러 개를 한번에: x1, x2, x3 = a1, a2, a3
    - 변수와 값의 개수가 동일해야 함
    - 변수의 값이 모두 같을 경우: x1 = x2 = x3 = a
  - 두 변수의 값 바꾸기: x1, x2 = x2, x1
4. 변수 삭제하기: del x
5. 빈 변수: x = None

6. 연산 후 할당: +=, -=, *=, /=, //=, %=
7. 부호 붙이기: +x, -x

8. input(): 사용자가 입력한 값을 가져오는 함수
  - 프롬프트: 괄호 안에 넣는 문자열
    - 사용자에게 입력받는 값의 용도를 미리 알려줌
  - 결과값은 문자열
  - 한번에 값 여러개 입력받기: input().split('기준문자열')
  - 정수로 변환: map(int, input().split('기준문자열'))
    - 실수로 변환 시 int 대신 float
